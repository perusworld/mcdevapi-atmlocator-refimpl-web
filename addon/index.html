API Demo Addons

Click on the link below for the browser you want to install on, it should ask you permission to install the addon.

Firefox

If for some reason it does not allow you to install, just right click on the file and save it locally. Then open Firefox, drag and drop the downloaded file on to firefox window to install.

Testing

To test the app, open the browser you want to test on and go to the Locations API Documentation page @ https://developer.mastercard.com/documentation/locations/

Once the page loads, after about 5 seconds scroll down to What you'll need section. You should see a new Reference Implementation section. If it doesnot appear the first time, just refresh the page.
